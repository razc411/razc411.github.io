    What Have I Done?


    Cut The Power


    The Project

    A 32 player networked video game in the style of Cops and Robbers.

    In Depth

    This game was written in C using an entity object system instead of an object oriented scheme. It proved to have great benefits in some areas over tradional Object Oriented programming. The best of which, was object independency.

    My Part

    I handled the client side networking interactions of Cut the Power. In doing this, I dealt with game to network interactions and debugging interactions with the game server.

    LxIDS


    The Project

    A simple, configurable intrusion prevention system written in Ruby for Linux.

    In Depth

    This project is configurable to any sort of authentication event you wish to watch. It handles IP banning of hosts being too forceful on your network.

    My Part

    This project was a work I did alone, I found it quite a breath of fresh air to get my hands on Ruby. The simple object oriented scheme of Ruby made it feel as if I was making pseudocode that was code itself.

    QtTCP


    The Project

    A multi-threaded file transfer client built with C++ and the Qt framework.

    In Depth

    The project was created to transfer multiple files simultainiously over TCP, within I implemented a circular buffer and a threaded system which allowed for multiple downloads to occur at a time.

    My Part

    I created this project myself, it was a re-creation of many pro,rams ive made before as file transfer client/servers. I found the Qt framework made working with C++ quite a bit more enjoyable, though I have my gripes about the simplicity of the network API.

    GPS Runner


    The Project

    An Android application for runners. The project was created as a prototype for a business for their future application.

    In Depth

    It was created using Java and makes extensive use of the accelerometer and the GPS within the phone. The project showed me how difficult it can be to transfer real world feedback into usable data.

    My Part

    I worked on many parts of this, including the routing, GPS tracking and GUI. The project was completed with 2 colleagues and went over well in its presentation.
